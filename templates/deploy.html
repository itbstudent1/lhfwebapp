LHF Setup

To set up local or remote environment on Linux Ubuntu:

Docker must be installed https://docs.docker.com/install/linux/docker-ce/ubuntu/#install-using-the-repository

Commands to run:

docker run -d -p 80:80 itbstudent1/lhf-http

docker run -d -p 21:21 itbstudent1/lhf-ftp

** FTP service may require port opening on firewall for remote deployment **

iptables -A INPUT -p icmp --icmp-type echo-request -j REJECT

To run the LHF Webapp container local or remote:

docker run -d -p 80:80 itbstudent1/lhf-webapp

** change host port if in use e.g. -p 8001:80 **

Thats it!
